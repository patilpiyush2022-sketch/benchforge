FAQ

How BenchForge Works

Answers to common questions.

How are scores calculated?

BenchForge measures execution speed of CPU, memory, storage and graphics workloads and maps them to a 0–10,000 scale using heuristics intended to be comparable on the same browser engine. Results are approximate and intended for relative comparisons.

Are scores comparable?

Scores are best used for comparing systems running the same browser and version. Browser scheduling, background tasks, and power settings can affect results.
